# PortalIrmen

# 🏢 Portal Corporativo - Protótipo de Organograma

## 📌 Visão Geral

Este projeto consiste no desenvolvimento de um **protótipo de organograma corporativo**, com o objetivo de representar de forma clara e estruturada a organização da unidade, permitindo a visualização hierárquica e o acesso a informações básicas dos colaboradores.

A solução foi desenvolvida como uma **aplicação web estática**, hospedada via GitHub Pages, com foco em simplicidade, baixo custo e rápida validação do conceito.

---

## 🎯 Objetivo

O principal objetivo deste projeto é:

Desenvolver uma solução inicial para visualização do organograma corporativo e diretório de colaboradores, permitindo validar a estrutura, usabilidade e utilidade da ferramenta antes de uma possível evolução para um portal interno completo da empresa.

---

## 🚀 Funcionalidades do Protótipo

* Visualização do **organograma hierárquico**
* Diretório de colaboradores em formato de **cards**
* Busca por:

  * Nome
  * Setor
  * Base regional
* Filtros por área/setor
* Estrutura organizada por gestor direto
* Interface simples e intuitiva

---

## 🧱 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* JSON (estrutura de dados dos colaboradores)

## ⚠️ Considerações de Segurança

Este projeto foi desenvolvido como **protótipo de validação**, portanto:

* Não contém dados sensíveis
* Não possui controle de acesso
* Está hospedado em ambiente público (GitHub Pages)

Para uso corporativo real, recomenda-se:

* Migração para ambiente interno (intranet)
* Implementação de controle de acesso
* Integração com base de dados corporativa
